# 🎱 8-Ball Pool Score Tracker

Ek simple aur mobile-friendly web app jo pool race games ke scores track karne ke liye hai.

## ✨ Features

- ➕ Player add/remove karein
- 🔢 Scores adjust karein: **+2, +3, +4, +5, +6, +7, +10** aur **-2 se -10** tak
- 🔻 **Negative scores** allowed hain
- 🏆 Live standings aur stats (leader, total points, highest score)
- 💾 Data **IndexedDB** mein locally save hota hai (browser cache clear na karein toh data rahega)

## 🎯 Kaise Use Karein?

1. Player name daalein aur **Add** button click karein.
2. Player card ke neeche **+** ya **−** buttons se score badhayein/ghatayein.
3. Standings section mein ranking live update hoti hai.
